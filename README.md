# 🌐 UniMax

**UniMax** is a modern and responsive **Service Agency Website** built with the latest technologies including **Next.js 15**, **Tailwind CSS 4**, and **Shadcn UI**. It features a clean layout, animated transitions, dark mode, and a component-based UI system designed for scalability.

---

## 🚀 Live Preview

👉 [Live Site](https://unimax.vercel.app)

---

## ✨ Features

- ⚡ Built with Next.js 15 and Turbopack
- 🎨 Tailwind CSS 4 for utility-first styling
- 🧱 UI Components via Shadcn UI
- 🌙 Dark/Light mode toggle using `next-themes`
- 🎞️ Smooth animations powered by Framer Motion
- 🧩 Class name composition with `clsx`, `tailwind-merge`, and `cva`
- 🔔 Toast notifications using Sonner
- 💅 Prettier with Tailwind plugin for formatting
- 🔒 NextAuth.js setup-ready for authentication
- ✅ TypeScript & ESLint for better developer experience

## 📁 Project Structure

```
unimax/
├── app/                    # App directory (Next.js App Router)
├── components/             # Custom + Shadcn components
├── public/                 # Static files (images, icons)
├── styles/                 # Global styles
├── .gitignore             # Git ignored files
├── .prettierignore        # Prettier ignored files
├── eslint.config.mjs      # ESLint configuration
├── next.config.ts         # Next.js config

```

## 🧑‍💻 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/muhammadsaif7717/unimax.git
cd unimax
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Run the development server

```bash
npm run dev
```

Open your browser at [http://localhost:3000](http://localhost:3000) to view the site.

---

## 🏗️ Build for Production

```bash
npm run build
npm run start
```
